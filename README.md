# 🌌 Schwarzschild Photon Sphere Simulator

A Python-based scientific visualization project that simulates photon trajectories around a Schwarzschild Black Hole.

The project demonstrates:

- ⚫ Black Hole Event Horizon
- 🔴 Photon Sphere
- 🟠 Accretion Disk
- 🔵 Curved Photon Trajectories
- ✨ Space Background with Stars
- 📈 Scientific Visualization using Matplotlib

---

## Preview

This simulation visualizes how light behaves near a black hole and highlights the concept of a photon sphere, where photons can orbit due to extreme gravity.

## Physics Background

For a Schwarzschild Black Hole:

Schwarzschild Radius:

rₛ = 2GM/c²

Photon Sphere Radius:

rₚ = 3GM/c²

The photon sphere is the region where gravity is strong enough for photons to travel in unstable circular orbits.
